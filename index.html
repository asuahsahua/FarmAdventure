##Feasibility testing
## Feasibility Study * Single Player * Multiplayer - Cooperative * Multiplayer - Competitive

Criteria

  • Connectivity and bandwidth
  • Server infrastructure costs
  • Development time expectations
## Methods * Connectivity and bandwidth * Server infrastructure * Development time

Evaluation - Single Player

  • Connectivity and Bandwidth
    • None
  • Server infrastructure costs
    • None
  • Development time expectations
    • 120 hours

Evaluation - Cooperative Multiplayer

  • Connectivity and Bandwidth requirements
    • 0.9 KB/s upstream, 1.65 KB/s downstream
  • Server infrastructure costs
    • Can be run on a free Heroku server
  • Development time expectations
    • 195 hours

Evaluation - Competitive Multiplayer

  • Connectivity and Bandwidth requirements
    • Significantly higher than Cooperative Multiplayer
  • Server infrastructure costs
    • Cannot be run on a free Heroku Server
  • Development time expectations
    • >195 hours
